# Sensor Fault Detection using Machine Learning

## Overview
This project focuses on detecting faults in sensor data using machine learning techniques. It analyzes sensor readings to identify abnormal patterns and predict potential failures.

## Features
- Data preprocessing including handling missing values and outliers
- Exploratory data analysis to understand data patterns
- Training and evaluation of machine learning models
- Fault prediction based on sensor input
- Web application built using Flask to take user input and display results

## Technologies Used
- Python
- Pandas and NumPy
- Scikit-learn
- Flask

## Model Details
Different machine learning models were tested and evaluated using metrics such as Mean Squared Error (MSE) and R² score. The best performing model was selected based on accuracy and reliability.

## How to Run
1. Clone the repository  
2. Install the required libraries  
3. Run the Flask application  
4. Enter sensor values through the web interface to get predictions  

## Project Structure
- data preprocessing and analysis
- model training and evaluation
- Flask application for user interaction

## Conclusion
The project demonstrates how machine learning can be applied to detect faults in real-world sensor data and provide predictions through a simple web interface.
